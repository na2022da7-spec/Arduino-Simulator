# Arduino-Simulator

## Description
This project controls four servo motors using Arduino. The motors start at the center position (90°), move between 40° and 140° three times, then return to the center.

## Requirements
- Arduino Uno
- 4 Servo Motors


## How to Run
1. Connect the four servo motors to pins 3, 5, 6, and 9.
2. Upload the code to the Arduino board.
3. Power the circuit.
4. The servo motors will perform the programmed movement automatically.

## Output
- All four servo motors move together from 90° to 40°, then to 140°, repeating three times before returning to the center position.
